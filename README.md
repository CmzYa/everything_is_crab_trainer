##  安装方法（手动安装）

### ⭐ 推荐方式：直接复制到游戏根目录

#### 步骤 1：找到游戏目录

打开游戏所在的文件夹，你应该能看到 `Everything.exe` 文件。

**常见位置示例：**
```
D:\Steam\steamapps\common\EverythingIsCrab\
```
或
```
C:\Program Files (x86)\Steam\steamapps\common\EverythingIsCrab\
```

#### 步骤 2：复制所有文件

**将本压缩包内的所有文件和文件夹，直接复制粘贴到游戏根目录！**

**✅ 正确的文件结构（安装后）：**
```
📁 游戏根目录（Everything.exe 所在位置）
│
├── 📄 Everything.exe                    ← 游戏主程序（已存在）
│
├── 📁 BepInEx/                         ← 从压缩包复制的整个文件夹
│   ├── 📁 core/
│   ├── 📁 plugins/
│   │   └── 📁 EverythingIsCrabMod/
│   ├── 📁 config/
│   ├── 📁 interop/
│   └── （首次运行后生成 LogOutput.log）
│
├── 📄 doorstop_config.ini              ← 从压缩包复制
├── 📄 winhttp.dll                      ← 从压缩包复制
│
└── 📄 README.md                        ← 本文件（可选保留）
```

#### ❌ 常见错误：

**错误示例 1：创建了多余的子文件夹**
```
❌ 游戏目录\BepInEx_Cracked\BepInEx\...    ← 错误！多了一层！
✅ 游戏目录\BepInEx\...                     ← 正确！
```

**错误示例 2：只复制了部分文件**
```
❌ 只复制了 BepInEx 文件夹，忘记复制 doorstop_config.ini 和 winhttp.dll
✅ 必须复制所有文件和文件夹
```

#### 步骤 3：确认安装成功

检查以下文件是否都在**游戏根目录**：
- [ ] `BepInEx` 文件夹（整个文件夹）
- [ ] `doorstop_config.ini` 文件
- [ ] `winhttp.dll` 文件

如果都在，说明安装成功！✅

## 🎮 使用方法

### 显示/隐藏修改器

游戏中按下以下任一键：

- **F9** 键
- 或 **Insert** 键  
- 或 **Home** 键

### 功能列表（7 个标签页）

| 标签页 | 主要功能 |
|--------|----------|
| 🎯 **玩家** | 生命值、移速、伤害、无敌模式 |
| 🧬 **进化** | 进化等级、点数、基因修改 |
| ⚙️ **配置** | 难度、种子、教程跳过 |
| ⏱️ **时间** | 时间加速/减速、暂停 |
| 💊 **状态** | Buff/Debuff 管理 |
| 📊 **元进度** | 成就解锁、统计修改 |
| ⚡ **功能** | 一键操作按钮 |

### 常用操作示例

**锁定满血：**
1. F9 打开修改器 → 「玩家」标签
2. 找到「生命值」→ 勾选「锁血」
3. 设置血量数值

**无敌模式：**
1. 「玩家」标签 → 勾选「无敌」

**时间加速：**
1. 「时间」标签 → 调整倍率滑块
2. 2.0x = 双速，0.5x = 半速

**一键解锁成就：**
1. 「元进度」标签 → 点击「解锁全部成就」

## ❓ 常见问题

### Q: 修改器不显示？
**A:** 
1. 确认文件放在**游戏根目录**（不是子文件夹）
2. 按 F9/Insert/Home 切换显示
3. 检查杀毒软件是否拦截（添加白名单）
4. 查看 `BepInEx\LogOutput.log` 日志

### Q: 游戏崩溃？
**A:** 
1. 确保版本匹配
2. 删除 `BepInEx` 文件夹重装
3. 查看日志文件排查错误

### Q: 被杀毒软件报毒？
**A:** 正常现象！修改器会被误报。
- Windows Defender: 添加排除项
- 第三方杀软: 添加信任/恢复文件

### Q: 如何卸载？
**A:** 删除这 3 个东西即可：
- `BepInEx` 文件夹
- `doorstop_config.ini`
- `winhttp.dll`
